Guia de Configuracion

Opcion A — Google Apps Script (recomendado)
  1. Ve a script.google.com y crea un proyecto nuevo.
  2. Pega el codigo Apps Script (ver al final de este archivo HTML).
  3. Cambia SPREADSHEET_ID por el ID de tu hoja.
  4. La hoja debe tener columnas: ID | Nombre | Descripcion | Estado | Creada | Actualizada | Asignado
  5. Ejecuta initSheet() una vez para crear los encabezados.
  6. Implementar → Nueva implementacion → Aplicacion web. Ejecutar como: Yo. Acceso: Cualquiera.
  7. Pega la URL en Configuracion → Conectar.

ID de la hoja: la parte de la URL entre /d/ y /edit.
Ej: docs.google.com/spreadsheets/d/1BxiMVs0XRA5.../edit
