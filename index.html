Guia de Configuracion

Opcion A — Google Apps Script (sin OAuth, recomendado)
  1. Ve a script.google.com y crea un proyecto nuevo.
  2. Borra el contenido y pega el codigo que esta al final del archivo dashboard-tareas.html.
  3. Cambia SPREADSHEET_ID por el ID de tu hoja de calculo.
  4. Tu hoja debe tener una pestana llamada Tareas con la fila de encabezados:
    ID | Nombre | Descripcion | Estado | Creada | Actualizada
  5. Haz clic en Implementar → Nueva implementacion → Aplicacion web.
  6. Ejecutar como: Yo. Acceso: Cualquiera. Autoriza y copia la URL.
  7. Pega la URL en el campo "URL del Web App" y haz clic en Conectar.
Opcion B — Sheets API + OAuth
  1. Ve a console.cloud.google.com.
  2. Crea un proyecto, activa Google Sheets API.
  3. En Credenciales, crea un ID de cliente OAuth 2.0 → Aplicacion web.
  4. Agrega el origen donde abres este HTML como "Origen de JavaScript autorizado".
    (Ej: http://localhost:8080 si lo sirves localmente)
  5. Copia el Client ID. Ingresalo junto con el Spreadsheet ID y haz clic en Conectar.

El ID de la hoja de calculo es la parte de la URL entre /d/ y /edit.
Ej: docs.google.com/spreadsheets/d/1BxiMVs0XRA5.../edit
